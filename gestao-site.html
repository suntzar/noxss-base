Bem-vindo ao Sistema de Gestão Escolar

Este é um template inicial para construir aplicações web modernas e responsivas. Navegue pelas abas para ver exemplos de conteúdo e componentes.

Sobre a Biblioteca Noxss

A Noxss é uma biblioteca CSS e JS com filosofia "App-First", projetada para resolver problemas comuns no desenvolvimento de interfaces, como:

  • Layout de Aplicativo: Estrutura pronta para interfaces de tela cheia com navbars e conteúdo rolável.
  • Tematização: Suporte nativo para temas Light/Dark e customização via Variáveis CSS.
  • Componentes Funcionais: Um conjunto de componentes reutilizáveis como botões, modais, e abas, prontos para uso.

O objetivo é fornecer um ponto de partida sólido para que você possa focar na lógica do seu aplicativo, e não na estrutura básica da UI.
